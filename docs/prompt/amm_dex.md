- このプロンプトを実行する前に以下の2つのMCPを起動しておくこと
  - context7 
  - deepwiki

```markdown

- 必要な画面一覧
- 必要な機能一覧
- 採用する技術スタック

UNCHAINで紹介している以下の学習コンテンツで作成できるような完成度のものをイメージしています。
```
